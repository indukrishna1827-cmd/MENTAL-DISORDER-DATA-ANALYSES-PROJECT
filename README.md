Mental Disorder Analysis Project

Exploratory Data Analysis (EDA) & Interactive Power BI Dashboard

 Overview

This project focuses on analyzing mental health disorder–related data to uncover patterns, trends, and insights.
The workflow includes:

Performing Exploratory Data Analysis (EDA) in Google Colab

Cleaning, transforming, and visualizing data

Building an interactive Power BI dashboard for intuitive exploration

Highlighting key findings that can support further research or awareness efforts


├── notebooks/             # Colab EDA notebooks

├── powerbi/               # Power BI files (.pbix)

├── images/                # Dashboard screenshots 

└── README.md              # Project documentation

Exploratory Data Analysis (EDA)

Key steps performed in the notebook include:

Data loading & attribute description

Handling missing values

Detecting outliers

Feature analysis (univariate, bivariate, multivariate)

Correlation heatmaps

Visualization of mental disorder trends and relationships


Power BI Dashboard

An interactive Power BI dashboard was created to make insights easy to explore.
Dashboard features include:

Filters and slicers for demographic segments

Trends in disorder prevalence

Visualization of symptom patterns

Comparative analyses

Highlighted KPIs and summary metrics

Insights / Key Findings

Some insights explored include:

Most common mental disorders across demographic groups

Factors associated with higher disorder prevalence

Distribution of symptoms and severity


Technologies Used

Python (Google Colab)

pandas, numpy
